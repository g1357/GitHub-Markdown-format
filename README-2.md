# "Элемент управления Scene3DControl

Элемент управления Scene3DControl предназначен для отображения 3D-сцены.
Он может быть использован в любом месте, где требуется отображение 3D-графики,
например, в окне приложения, панели инструментов или другом элементе управления.

Для использования Scene3DControl необходимо добавить его в разметку XAML и настроить его свойства.

## Настройки
- **AutoZoomOnAdd**: Если установлено в true, то при добавлении нового объекта в сцену, 
  камера автоматически будет приближаться к нему. (По умолчанию: true)
- AutoZoomOnResize: Если установлено в true, то при изменении размера элемента управления, 
  камера автоматически будет подстраиваться, чтобы все объекты в сцене были видны.
  (По умолчанию: true)
- SceneMarginRatio: Устанавливает соотношение между размером сцены и размером элемента управления. 
  Это позволяет контролировать, насколько близко или далеко объекты будут отображаться. 
  (По умолчанию: 0.05 - 5%)
